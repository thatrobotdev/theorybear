## What is a staff?

<figure class="card card__body">

<div class="darkmodeWhiteBackground">
  <StaffSVG 
    alt="A typical five-line staff with five equally spaced horizontal lines on a white background"
    className="card__image fillWhite"
  />
</div>

<figcaption>
      A typical five-line staff
      <footer class="card__footer">
        <a href="https://commons.wikimedia.org/wiki/File:Staff240.svg">っ</a>, <a href="http://creativecommons.org/licenses/by-sa/3.0/">CC BY-SA 3.0</a>, via Wikimedia Commons
      </footer>
  </figcaption>
</figure>

One way to think about a staff (plural "staves") is that it is an empty canvas on which we can draw notes. Just like how the alphabet allows you to write words, the staff allows you to write notes!

> A staff consists of five horizontal lines, evenly spaced, and each note is placed on the line or space that corresponds to its pitch.
<footer>—"<a href="https://viva.pressbooks.pub/openmusictheory/chapter/notation-of-notes-clefs-and-ledger-lines/#chapter-29-section-2">Notation of Notes, Clefs, and Ledger Lines: Staff Notation</a>" from <a href="https://viva.pressbooks.pub/openmusictheory/">OPEN MUSIC THEORY</a> by <a href="https://cnu.edu/people/chelseyhamm/">Chelsey Hamm</a>, licensed under <a href="https://creativecommons.org/licenses/by-sa/4.0/">CC BY-SA 4.0</a></footer>

You count lines and spaces of a staff from the bottom. The bottom line is line 1, the space above it is space 1, the line above that is line 2, and so on.
